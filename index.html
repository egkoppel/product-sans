Product Sans Webfont

To add the font to a webpage, copy this code into the <head> of your webpage.
<link rel="stylesheet" href="https://egkoppel.github.io/product-sans/google-fonts.css">

Then add id="product-sans" to the text.

You can insert a google logo, which looks like this:

google_logo

To insert it, while using product sans, type "google_logo".

You can also insert the Google G, which looks like this:

google_G

To insert it, while using product sans, type "google_G".
